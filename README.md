# 简介：

批量寻找可白嫖的百度云盘在线解析

------

# 快速使用

## URL文件

1.通过各种手段收集到的url写入url.txt文件中（每行一条url）。

2.工具主页面选择url列表文件。

## FOFA

1.配置自己fofa帐号的email和api_key即可，查询数据大小（saiz）根据帐号权限设置。

2.工具主页面选择FOFA
